## 安装 ThinkJS

```sh
npm install -g thinkjs-cli
```

## 创建项目

```sh
thinkjs new project_path --es6
```

## 安装依赖

```sh
npm install
```

## 启动服务

```sh
npm start
```

启动后，会看到类似下面的信息：
